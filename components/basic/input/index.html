    API

    通用参数

    参数说明类型默认值
    showOpt是否显示 input 下方的确认及关闭按钮booleanfalse
    errorMsg错误信息，如果填写了，会在 input 下方显示string-
    labellabel 信息，如果填写了，会在 input 上方显示string-
    wrapStyleInput 组件外层容器的 styleReact.CSSProperties-
    onOk确认按钮点击回调Function-
    onCancel关闭按钮点击回调Function-

    Input

    参数说明类型默认值
    value值string必填
    onChange修改值时触发的回调函数Function(e)必填
    type同原生 input 的 typestring'text'
    onPressEnter按下回车时的回调Function(e)-
    disabled是否禁用booleanfalse
    maxLength输入文字的最大长度number-
    placeholder默认显示文本string-
    inputStyleinput 的 styleReact.CSSProperties-
    ...支持其他 React 自带的 input 属性any-
